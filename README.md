## Como Executar o Projeto
1. Clone este repositório:
```
git clone <URL_DO_REPOSITORIO>
```
2. Navegue até o diretório do projeto:
```
cd nome-do-projeto
```
3. Execute o comando principal:
```
./iniciar.sh
```
